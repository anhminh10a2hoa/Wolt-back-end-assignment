# Wolt Backend Assignment 2021

## How to run

+ Clone my repo: 

```batch
git clone https://github.com/anhminh10a2hoa/Wolt-front-end-assignment
```

+ Install packages and its dependencies:

```
npm install
```

+ Run this assignment:

```
npm start
```

+ Run the API on port 8000:
```
http://localhost:8000/discovery?lat=60.1709&lon=24.941
```
